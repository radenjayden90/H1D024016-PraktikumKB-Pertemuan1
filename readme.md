# Praktikum KB Pertemuan 1

Program Python sederhana untuk menampilkan dan merekomendasikan menu makanan.

## Fitur Program
1. Melihat daftar menu makanan
2. Menampilkan rekomendasi menu secara acak
3. Keluar dari program

## Konsep yang Digunakan
- Struktur kontrol (if, while, for)
- Struktur data (list)
- Library Python (random, datetime)

## Bahasa Pemrograman
Python
